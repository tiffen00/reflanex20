⚙️ Configuration

Chargement…

🔍 Trouver ton ID Telegram : Envoie un message à @userinfobot et il te donnera ton ID.

➕ Ajouter un admin : Ajoute son ID Telegram dans la variable TELEGRAM_ADMIN_IDS sur Render (séparés par virgule) et redéploie.

🆘 Dépannage

Le bot ne répond pas ?

  • ✅ TELEGRAM_BOT_TOKEN est bien configuré sur Render ?
  • ✅ TELEGRAM_ADMIN_IDS contient bien ton ID Telegram ?
  • ✅ Le service Render est démarré (vérifie le statut "Live") ?
  • ✅ Tu as bien envoyé /start au bot au moins une fois ?
  • ✅ Le token n'est pas révoqué ? (Recrée-le avec @BotFather si besoin)

📋 Voir les logs : Va sur Render → ton service → Logs pour voir les erreurs en détail.
